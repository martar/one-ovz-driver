Welcome to OpenNebula driver for OpenVZ hypervisor

This set of drivers allows you to use OpenVZ as a hypervisor to deploy virtual machines in OpenNebula.

Description

One-ovz-driver consists of IM_MAD scripts that are able to monitor the remote hosts and a VMM_MAD scripts to deploy and manage Virtual Machines. VMM_MAD serie's scripts make use of OpenVZ API

Benefits

Extending OpenNebula system by including a well known container-based virtualization for Linux - OpenVZ.

Cluster Node requirements

  1. ruby >= 1.9

    • rvm is recommended to manage ruby version
    • ex. rvm tutorial for centos 6

  2. rake

    • should be available by default, if not then install them manually
$ gem install rake
  1. remaining gems
$rake gems
